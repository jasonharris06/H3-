# Trivia Game
## Overview 
This is a Rick and Morty Trivia game.  The user will guess 10 questions with 4 possible answers in 60 seconds.  After the time is up the correct answers will be totaled up giving the user their score.  

### Coding Skills utilized in assignment
-	**SetTimeout** was used to create a timer that would count down to zero.  This timer was displayed on the screen.  
-	**Creating questions with radio buttons** nested for loops were used to create dynamically created list from preset questions and possible answers.
